# Prédiction de Pourboires de Taxis à NYC avec XGBoost

## Description du Projet

Ce projet vise à prédire le montant du pourboire (tip amount) pour les courses de taxis jaunes à New York City (NYC), en utilisant l'algorithme de régression XGBoost.
Le code est optimisé pour les grands ensembles de données via l'utilisation de cuDF/Pandas (traitement GPU).

## Jeu de Données

Le dataset de travail a été simplifié à partir des données publiques du "2023 Yellow Taxi Trip Data" de la ville de New York.
